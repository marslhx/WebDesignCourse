syntax of CSS

Declaration is the green part below:

selector{

property1:value;

property2:value;

{

Element selectors like this:

body { background: chartreuse; }

Class selector like this:

.Declaration { color:green; }

Id selectors like this:

#comments { color:green; }

Group selectors like this:

span, div { font-family:monospace }

Pseudo-class selectors like this:

.cases:hover{ background:salmon }

A comment in CSS looks like /* this is a comment */
