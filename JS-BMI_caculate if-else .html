About standard weight(ST):

For man:
ST=height-100.
if (weight-ST)>3, overweight;
if (weight-ST) <-3, underweight;
otherwise, normal.

For woman:
ST=height-110.
if (weight-ST)>3, overweight;
if (weight-ST) <-3, underweight;
otherwise, normal.

Input "gender","height" and "weight" in input box，Click “Appraise” button to check the result.
